# Quickstart

## 1) Run the fixer

```bash
chmod +x fix.sh
./fix.sh
```

## 2) Recommended first run

- **1 — Full Auto Fix** (safe defaults)
- **4 — Install Essentials (Minimal)**
- **7 — Diagnose & Create Report** (if anything still fails)

## 3) Where logs & reports live

- Logs: `~/.termux-fixer/termux-fixer-*.log`
- Reports: `~/.termux-fixer/diagnose-*.txt`

Tip: When opening a GitHub issue, attach the report (remove anything private first).
