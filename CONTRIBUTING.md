# Contributing

Thanks for improving Termux-Fixer.

## How to contribute
- Open an issue with:
  - What you expected
  - What happened (copy/paste the error)
  - Your **diagnose report** from `~/.termux-fixer/diagnose-*.txt` (remove private info)
- Or open a PR with a focused fix.

## Style
- Keep changes **non-destructive** (never delete user data by default).
- Prefer small, composable functions.
- Add clear user-facing messages.
